Welcome to the Hilichurlian Language Database! This website is a repository for all utterances of the Hilichurlian language..

The database only contains official or "official" translations (in English) of Hilichurlian. Some educated guesswork was performed when deciding which utterances had translations provided, but aside from that, there is nothing unofficial in this database.

More features are on the way! Check the About page for more details.
