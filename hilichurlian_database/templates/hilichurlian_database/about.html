About

This website was created and is maintained by a Genshin Impact Fandom Wiki editor.

I created this website half because I believe hiliologists need it and half because I wanted to learn something new (namely, how to deploy a website using Heroku).

To send feedback about this database website, please join the Genshin Impact Wiki's Discord server and send a message in the #genshin-languages channel, which is viewable upon self-assigning the Lore role in the #librarians-rules channel. I promise I don't bite.
