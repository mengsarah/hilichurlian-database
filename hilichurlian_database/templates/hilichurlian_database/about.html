About

This website was created and is maintained by a Genshin Impact Wiki editor. This website has limitations on how long it can be up and running every month, so in return, the website "sleeps" if no page has been viewed for a while and then takes a few extra seconds to wake up when someone visits the website again.

I created this website half because I believe hiliologists need it and half because I wanted to learn something new (namely, how to deploy a website using Heroku).

To send feedback about this database website, please join the Genshin Impact Wiki's Discord server and send a message in the #genshin-languages channel, which is viewable upon self-assigning the Lore role.
